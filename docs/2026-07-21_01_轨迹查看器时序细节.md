# 轨迹查看器时序细节

轨迹查看器在原有关节角与 TCP 总览曲线的基础上，新增两个详细观察区域，均为本地只读显示，不修改 HDF5 或控制硬件。顶部可通过“上一条轨迹”和“下一条轨迹”按钮，按轨迹修改时间从新到旧切换；在最新或最旧轨迹处，对应按钮自动禁用。

## 关节角详细时序

详细区在 TCP 之后首先显示六关节的合并详细图，使用与总览一致的颜色图例，便于比较关节之间的联动。`J1` 至 `J6` 随后分别绘制在六张独立图中，每张图只根据该关节的数值范围设置纵轴。这样能观察到合并图中不易发现的小幅回差、抖动或突跳。标题、颜色标识和单位均直接标在对应图上，单位为 rad。

## TCP 详细时序

TCP 被拆分为两张图：

- `x`、`y`、`z`：位置，单位 m；
- `rx`、`ry`、`rz`：Piper 原生欧拉角，单位 rad。

位置和姿态不再共用纵轴，避免 m 与 rad 的量纲差异掩盖曲线细节。每张图上方的颜色图例与曲线一一对应；底部显示该序列的起止相对时间。

启动命令保持不变：

```bash
UV_CACHE_DIR=/tmp/uv-cache uv run piper-collect trajectory-viewer \
  --config configs/pick_up_the_corn_and_put_it_on_the_plate.yaml
```
